# ANRE Company Website

This is the **official company website** for ANRE Oy.
The website is a **single-page responsive site** built with HTML, CSS, and JavaScript, and is deployed live. It includes:

- **Two languages** (FI/EN) with a language toggle.  
- **Contact form** integrated with [EmailJS](https://www.emailjs.com/) to connect with the company directly via email.  
- Multiple sections including About Us, Services, and Call-to-Action (CTA).  
- Mobile-friendly, responsive design with smooth scrolling and modern UI elements.

> ⚠️ **Note:** Not all source code is included here due to privacy and company confidentiality.

You can view the live website here: [anre.fi](https://anre.fi)

---

## Tech Stack

- HTML5 & CSS3  
- JavaScript (Vanilla JS + jQuery for animations & UI interactions)  
- Responsive design for desktop and mobile  
- EmailJS for contact form integration  

---

## Features

- Fully responsive single-page design  
- Multi-language support  
- Smooth scrolling and interactive elements  
- Contact form with email integration  
- Clean, modern UI built from scratch
